# Personal Curriculum App

This project is designed to help users manage their learning journey with features like a Pomodoro timer, progress tracker, skill tree visualization, and more.

## Features
- Pomodoro Timer
- Progress Tracker
- Learning Streak
- Daily Goal Setting
- Skill Tree Visualization
- Knowledge Knapsack

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/alienfrommarsy/personal-curriculum.git
   ```
2. Navigate to the directory and install dependencies:
   ```bash
   cd personal-curriculum
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
